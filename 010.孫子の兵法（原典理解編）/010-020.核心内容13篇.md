
# 1.始計篇（しけいへん）

### 核心

戦う前に「勝てるかどうか」を決める分析フレーム。

### 重要要素（五事）

- **道**：理念・ビジョンへの共感

- **天**：タイミング・景気・技術潮流

- **地**：市場構造・立地・ポジション

- **将**：リーダーの資質

- **法**：制度・ルール・実行力


### 現代への適用

- 新規事業の**Go / No-Go判断**

- プロダクト立ち上げ前の**市場×組織×技術**評価  
    👉 _「やる前に勝算が数値化できない案件はやらない」_

# 2.作戦篇（さくせんへん）

### 核心

戦争は**長引くほど負ける**。

### ポイント

- 物資・人材・士気は有限

- 長期戦は組織を腐らせる


### 現代への適用

- **PoC → MVP → 撤退判断**を高速化

- 赤字覚悟の消耗戦を避ける  
    👉 _「続けられる戦い」ではなく「早く終わらせる戦い」_


# 3.謀攻篇（ぼうこうへん）

### 核心

**最善は戦わずして勝つこと**。

### 優先順位

1. 敵の計画を壊す

2. 同盟を分断する

3. 戦う

4. 城を攻める（最悪）


### 現代への適用

- 価格競争を避け、**ルール自体を変える**

- 競合と直接戦わず、**別市場・別軸で勝つ**  
    👉 _「正面衝突＝敗北リスク最大」_

### 共通思想

> **勝敗は実行前に8割決まっている**


# 4.軍形篇（ぐんけいへん）

### 核心

**「まず負けない形を作れ」**

### ポイント

- 勝ちは偶然でも、敗北は必然

- 守備が完成して初めて攻撃が成立


### 現代への適用

- 先に**技術的負債・運用破綻**を潰す

- セキュリティ／SRE／レビュー体制の整備  
    👉 _「スケール前に壊れない設計を」_


# 5.兵勢篇（へいせいへん） 

### 核心

**勢いは「構造」から生まれる**

### 重要概念

- **正**：定常オペレーション

- **奇**：非定常・イノベーション


→ 正と奇の組み合わせが無限の変化を生む

### 現代への適用

- 通常開発（正）＋実験的開発（奇）

- 自動化で人の勢いを増幅  
    👉 _「人を頑張らせない、仕組みで加速する」_

# 6.虚実篇（きょじつへん）

### 核心

**強いところは避け、弱いところを突け**

### 原則

- 敵が守る所＝実（攻めない）

- 敵が疎かにする所＝虚（攻める）


### 現代への適用

- 巨大企業の主力市場を避け、**ニッチを狙う**

- ユーザーの不満が放置されている部分に集中  
    👉 _「競争しない場所を見つける能力が最強」_


# 7.軍争篇（ぐんそうへん）

### 核心

**競争環境では「速さ」と「位置」が命**

### ポイント

- 先に動くと不利になることもある

- 主導権は「有利な場所」を取った側にある

### 現代への適用

- 早期参入より**有利なポジション取り**

- 競合の動きを見て**後出し最適解**を出す  
    👉 _「最速より最適」_

### 現代への適用

- 新規事業：裁量と実験を許す

- 危機対応：権限集中・即断  
    👉 _「平時と有事で同じ運営は失敗する」_

# 9.九変篇（きゅうへんへん）

### 核心

**原則は守れ、だが執着するな**

### ポイント

- 状況に応じて戦略を変えよ

- 変えてはいけないものと、変えるべきものを区別


### 現代への適用

- ピボット判断

- 技術選定・アーキテクチャの柔軟性  
    👉 _「一貫性 ≠ 固定」_

>ピボット（事業転換）の判断は、**市場ニーズとのズレ、事業計画からの乖離、プロダクトマーケットフィット（PMF）の欠如** などを主な基準とし、**仮説検証の繰り返しの中で、想定外の変化や改善が見込めない状況** で、**迅速かつ客観的に戦略の方向転換（ピボット）を決断する** こと==。


# 10.地形篇（ちけいへん）

### 核心

**環境は「利用するもの」**

### 地形の考え方

- 有利・不利は固定ではない

- 知っている側が常に有利

### 現代への適用

- 規制・業界慣習を逆手に取る

- 自社が慣れた技術領域に戦場を引き込む  
    👉 _「環境理解＝隠れた競争力」_


# 11.九地篇（きゅうちへん）

### 核心

**状況ごとにマネジメントは変えよ**

### 代表的な考え

- 余地のある状況：慎重に

- 逃げ場のない状況：結束を強める

### 現代への適用

- 新規事業：裁量と実験を許す

- 危機対応：権限集中・即断  
    👉 _「平時と有事で同じ運営は失敗する」_


# 12.火攻篇（かこうへん）

### 核心

**破壊的手段は制御できる時だけ使え**

### ポイント

- 火（破壊）は一時的に有効だが反動が大きい

- 状況・天候・タイミングを誤ると自滅する


### 現代への適用

- 強い値下げ・訴訟・炎上マーケの慎重運用

- リストラ・大規模改修の副作用管理  
    👉 _「勝てても、壊しすぎると次がない」_


# 13.用間篇（ようかんへん）

### 核心

**情報こそ最強の兵器**

### ポイント

- 事実・噂・内部情報の使い分け

- 情報は金より安く、被害は最大


### 現代への適用

- ユーザーリサーチ／競合分析

- ログ・データ・インサイト活用  
    👉 _「勘より情報、情報より解釈」_
