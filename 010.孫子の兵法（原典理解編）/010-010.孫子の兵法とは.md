
# 概要

**孫子の兵法**は

> **戦わず・消耗せず・最小コストで勝つための意思決定書**  
> です。

単なる戦争マニュアルではなく、  
**戦略・組織・判断・情報・リスク管理**の原理を扱います。


# 基本思想（3行要約）

- **勝敗は戦う前に決まる**

- **最善は戦わずして勝つこと**

- **情報と状況適応が力を上回る**

# 構成

- 全13篇
- 内容は以下に集約される

| 領域  | 何を扱うか     |
| --- | --------- |
| 戦略  | 勝てる条件の見極め |
| 組織  | 負けない体制    |
| 環境  | 戦場・状況の選択  |
| 情報  | 見えない優位    |
| リスク | 破壊の制御     |

# 孫子の有名な原則

- **彼を知り己を知れば百戦危うからず**

- **戦わずして人の兵を屈するは善の善なる者なり**

- **速戦即決**


👉 共通点は「感情ではなく判断」
