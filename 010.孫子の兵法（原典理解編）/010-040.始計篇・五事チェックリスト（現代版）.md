
# 1.道（理念・一体感）

**YES / NO で確認**

- なぜ「今これをやるのか」を全員が説明できるか

- 現場が腹落ちしていない指示が多くないか

- 離脱・形骸化は起きていないか


❌ NOが多い場合  
👉 _戦う前に内部崩壊する_

# 2.天（タイミング）

- 市場は「早すぎる」または「遅すぎる」状態ではないか

- 技術・法規・ユーザー成熟度は揃っているか

- 外部要因に賭けすぎていないか


❌ NOが多い場合  
👉 _正しい戦略でも時期が悪い_

# 3.地（戦場・構造）

- 競合の土俵で正面衝突していないか

- 自社が有利な領域に戦場を設定しているか

- 勝ち筋が構造的に説明できるか


❌ NOが多い場合  
👉 _努力しても報われない_


# 4.将（リーダー）

- 意思決定が遅延・ブレを起こしていないか

- 責任の所在は明確か

- 現場からの信頼はあるか


❌ NOが多い場合  
👉 _優秀な部下ほど先に抜ける_

# 5.法（制度・実行力）

- 決めたことが現場で実行されているか

- 評価・報酬は方針と整合しているか

- 例外だらけのルールになっていないか


❌ NOが多い場合  
👉 _戦略はスライドで終わる_


# 実践用・簡易スコアリング

- 各項目 **0〜2点**

- 合計 **10点満点**


**判断基準**

- 8–10点：実行可

- 5–7点：修正必須

- 0–4点：着手すべきでない
