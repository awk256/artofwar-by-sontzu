# 『孫子の五事で解く安全なAI活用の意思決定』
<br>

AIは、判断を速く・大量に行える技術です。  
しかし、**何をAIに任せてよいか**は、人が決めなければなりません。

**孫子は「判断を自動化する前に、人が決めるべき条件」を定義しているから**最も適した戦略的な指針書となります。。

どこで本書では、中国古典『孫子の兵法』の核心である  
**始計篇・五事（道・天・地・将・法）**　を軸に、**現代のAI文脈に翻訳**しています。

- `AIを使ってよい条件`
- `任せてよい判断の範囲`
- `止めるべきタイミング`
- `責任とガバナンスの設計`

ここで扱うのは、  AIを「賢くする方法」ではありません。
**AIを間違えにくく使うための判断の設計**です。

<img src="images/TheArtofWarbySunTzu.png" width="400" alt="図解Strandsエージェント徹底解説">


本サイト『**孫子の五事で解く安全なAI活用の意思決定**』は、筆者が AI活用の設計論を研究する過程で記録してきたメモをもとに構成しています。

これから**AI活用**を検討される方々にとって、本サイトが少しでも参考になれば幸いです。

なお、掲載内容には筆者の主観による要約や再構成が含まれており、ご利用にあたっては、ご自身の判断と責任にてお願いいたします。

また、本サイトの内容は今後も適宜、更新・修正される可能性があります。あらかじめご了承ください。
